Windows下快速添加Java环境变量

BurpSuite需要jre才能运行,固有此文

下载Java SE

Java Downloads (oracle.com)

对于BurpSuite高版本建议使用 java11,用其它的版本均有概率出现不同的差错

下载其中一个

安装

按照安装向导一路向下

添加Java至系统环境变量

计算机-右键菜单-属性-高级系统设置-环境变量

在环境变量-系统变量新建JAVA_HOME,CLASSPATH,并追加Path

变量名 值
JAVA_HOME C:\ENV\jdk11
Path %JAVA_HOME%\bin;
Path %JAVA_HOME%\jre\bin;

CMD在非管理员模式下不生效

​ 需要在用户变量的Path中也追加上述值

配置完成后重启CMD或Poweshell窗口

输入

java

java -version

javac

查看输出判断是否正常运行
