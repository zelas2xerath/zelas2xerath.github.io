开始

DIY是“Do It Yourself”的英文缩写，最初兴起于电脑的拼装，逐渐演绎成为一种流行生活方式。

在一开始的时候，DIY这用语主要是专门针对住房整修，庭园维护时，人们不想花费太多的费。找寻专业人士施工，而是利用自行购买或租用工具与购买来的材料，在闲暇时自行整修房屋的行为。

渐渐的DIY的概念也被扩及到所有可以自己动手做的事物上，例如自行维修汽车与家电产品，或甚至购买零件组装个人计算机等，没有特别明确的使用范围定义。

DIY的目的也由一开始时节省开销的理由，慢慢地演变成一种以休闲、发挥个人创意或培养嗜好的风气。

进阶(预更)

在配置选择中想要有更多自己的想法

CPU

  • Q&A

    Q：我应该如何在 Intel 和 AMD 的 CPU 同级别产品中进行选择？

    A：如果你有硬性的偏好（特别喜好 Intel（AMD）的 CPU），可以自行选择

  1. ​ 在确定预算的情况下，你应该首要选择 CPU，而后才是其它硬件

  2. ​ 确定预期需求而又对于预算不太敏感，可以拟确定多种 CPU 选择以形成最终的配置，探讨细节后，最终选择一个最佳项

  • Q&A

    Q1：如何选购当下 CPU 不同的代次？
    A1：通常来说，新代次的 CPU 不仅仅是增强了性能，往往还有一些重要的新特性，这些新特性是未来硬件的发展方向

    Q2：散片还是盒装？

    A2：看个人喜好，散片处理器主要是价格优势，但如果优势不大，那还不如买盒装版来得合适（部分盒装附有原厂散热器，相当不错的有AMD 棱镜散热器）

桌面CPU性能天梯图

主板

往往来说，在追求性价比的配置中，都会选择板 U 套装的形式。

对于追求超频的配置选择应该更为慎重的考虑

显卡

目前，RTX 、RX 、ARC ，红绿蓝三家强势竞争显卡市场，相关竞品空前绝目+

在价格允许的条件下，会优先考虑 RTX 产品，因为 CUDA、DLSS、光追的竞争力更强

需要注意的是显卡输出接口请在购买显示器线缆时确认（ 推荐DP

内存

国产内存厂商崛起，不再是 三星、海力士、镁光 垄断市场，内存创新史低

国产内存颗粒竞争力不凡，大大增加了选择

  1. 目前来说至少会从 16GB 容量双通道内存开始考虑
  2. 购买 RGB 灯条时，应考虑内存条高度对风冷散热器的影响
  3. 应当避免杂牌内存，及颗粒混用的产品
  4. 购买 2 条内存时，应该选择套条
  5. 后期加内存时，应当购买相同型号的套条

  • Q&A

    Q：要不要上 DDR5 ？

    A：目前 DDR5 的主要优势并不体现在游戏，还在普及中，是未来的趋势，目前仍可使用 DDR4

硬盘

三不原则

  • 不买二手

  • 不选杂牌

  • 不当怨种(不买怨种型产品)

目前机械硬盘价格稳定，各厂家相继减少出货量，SSD 创新低。大方向是高速大容量 SSD，国产存储大厂崛起，前景无限。

  • Q&A

    Q：要不要上 PICE 4.0 固态？

    A：其实用户感知不强，优势主要体现在一次操作海量数据的情况下，PCIE 3.0 对于大多数人来说仍然够用

电源

【硬件科普】如何合理科学的选择电源功率的大小？

散热硅脂/导热贴/清洁剂

大部分 板U 或盒装已经附赠，无须额外购买硅脂

平民级产品可以考虑 7921 硅脂

部分显卡产品在长期高压工作中会”出油”，可以在量取宽高厚度之后选购导热贴

清洁电子产品时应额外慎重水洗，可以使用专用的有机溶剂洗涤
